# Textual Editor

## Install

Clone:

```sh
git clone https://github.com/kyrlian/textual-editor.git
cd textual-editor
```

Install and run with Poetry:

```sh
poetry install
poetry shell
ted <file name>
```

Install and run with pipx:

```sh
pipx install .
ted <file name>
```

uninstall from pipx:
```sh
pipx uninstall textual_editor
```
